# RobLix

A social game creation platform where users build and play 3D experiences together. Built on ProGameStore with real-time multiplayer via Cloudflare Durable Objects.

**Live:** https://roblix.progamestore.online

## Features

- 3D world with blocky avatars (Roblox-style characters)
- Third-person camera, WASD movement, space to jump
- Real-time multiplayer — see other players moving in the same world
- Chat overlay for communication
- World browser — pick an experience to join
- Avatar customization — color picker for head, torso, arms, legs

## Development

```bash
pnpm install
pnpm dev
```

## Deploy

```bash
pnpm deploy
```
